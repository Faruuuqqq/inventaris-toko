───────────────────────────────────────────────────────────────────────────────
1. MASTER DATA (5 Modul)
───────────────────────────────────────────────────────────────────────────────

✅ SUPPLIER
   • Tampilkan daftar supplier    → GET /master/suppliers
   • Tambah supplier              → POST /master/suppliers
   • Edit/Hapus supplier          → PUT/DELETE /master/suppliers/{id}
   • AJAX dropdown                → GET /master/suppliers/getList

✅ CUSTOMER  
   • Tampilkan daftar customer    → GET /master/customers
   • Tambah customer              → POST /master/customers
   • Customer dengan piutang      → GET /info/saldo/receivable
   • Edit/Hapus customer          → PUT/DELETE /master/customers/{id}
   • AJAX dropdown                → GET /master/customers/getList

✅ PRODUK
   • Daftar produk                → GET /master/products
   • Tambah/Edit/Hapus produk     → POST/PUT/DELETE /master/products
   • (Kategori termasuk field)    → Part of /master/products

✅ GUDANG
   • Daftar gudang                → GET /master/warehouses
   • Tambah/Edit/Hapus gudang     → POST/PUT/DELETE /master/warehouses
   • AJAX dropdown                → GET /master/warehouses/getList

✅ SALES
   • Daftar sales                 → GET /master/salespersons
   • Tambah/Edit/Hapus sales      → POST/PUT/DELETE /master/salespersons
   • AJAX dropdown                → GET /master/salespersons/getList

───────────────────────────────────────────────────────────────────────────────
2. TRANSAKSI (9 Jenis)
───────────────────────────────────────────────────────────────────────────────

✅ PEMBELIAN
   • Beli barang ke pihak lain    → GET/POST /transactions/purchases
   • Terima barang                → GET/POST /transactions/purchases/receive
   • Edit/Hapus pembelian         → PUT/DELETE /transactions/purchases/{id}

✅ PENJUALAN TUNAI
   • Buat transaksi tunai         → GET/POST /transactions/sales/cash
   • Get produk (AJAX)            → GET /transactions/sales/getProducts

✅ PENJUALAN KREDIT
   • Buat transaksi kredit        → GET/POST /transactions/sales/credit
   • Get produk (AJAX)            → GET /transactions/sales/getProducts

✅ PEMBAYARAN UTANG
   • Bayar utang ke supplier      → GET/POST /finance/payments/payable
   • Get invoice supplier (AJAX)  → GET /finance/payments/getSupplierPurchases

✅ PEMBAYARAN PIUTANG
   • Terima bayar dari customer   → GET/POST /finance/payments/receivable
   • Get invoice customer (AJAX)  → GET /finance/payments/getCustomerInvoices

✅ RETUR PEMBELIAN
   • Buat retur ke distributor    → GET/POST /transactions/purchase-returns
   • Approval retur               → GET/POST /transactions/purchase-returns/approve

✅ RETUR PENJUALAN
   • Buat retur dari customer     → GET/POST /transactions/sales-returns
   • Approval retur               → GET/POST /transactions/sales-returns/approve

✅ SURAT JALAN (DELIVERY NOTE)
   • Buat surat jalan             → GET/POST /transactions/delivery-note
   • Get items dari invoice       → GET /transactions/delivery-note/getInvoiceItems
   • Print surat jalan            → GET /transactions/delivery-note/print/{id}

✅ KONTRA BON
   • List bon (filter customer)   → GET /finance/kontra-bon
   • Buat kontra bon              → POST /finance/kontra-bon/store
   • Update status                → POST /finance/kontra-bon/update-status
   • Get bon list (AJAX)          → GET /finance/payments/getKontraBons
   • Export PDF                   → GET /finance/kontra-bon/pdf/{id}

3. INFORMASI / HISTORY (7 Tipe)
───────────────────────────────────────────────────────────────────────────────

✅ PEMBELIAN HISTORY
   • Histori pembelian            → GET /info/history/purchases
   • Data (AJAX)                  → GET /info/history/purchases-data
   • Export CSV                   → GET /info/history/purchases-export

✅ PENJUALAN HISTORY (+ HIDE FEATURE)
   • Histori penjualan            → GET /info/history/sales
   • Data (AJAX)                  → GET /info/history/sales-data
   • Toggle hide/show ⭐          → POST /info/history/toggleSaleHide/{id}
   • Export CSV                   → GET /info/history/sales-export

✅ RETUR PEMBELIAN HISTORY
   • Histori retur pembelian      → GET /info/history/return-purchases
   • Data (AJAX)                  → GET /info/history/purchase-returns-data

✅ RETUR PENJUALAN HISTORY
   • Histori retur penjualan      → GET /info/history/return-sales
   • Data (AJAX)                  → GET /info/history/sales-returns-data

✅ BIAYA/JASA HISTORY
   • Histori biaya                → GET /info/history/expenses
   • Data (AJAX)                  → GET /info/history/expenses-data
   • Management biaya             → GET /finance/expenses
   • Analisis biaya               → GET /finance/expenses/analyze-data
   • Export CSV                   → GET /finance/expenses/export-csv

✅ PEMBAYARAN UTANG HISTORY
   • Histori bayar utang          → GET /info/history/payments-payable
   • Data (AJAX)                  → GET /info/history/payments-payable-data
   • Export CSV                   → GET /info/history/payments-payable-export

✅ PEMBAYARAN PIUTANG HISTORY
   • Histori bayar piutang        → GET /info/history/payments-receivable
   • Data (AJAX)                  → GET /info/history/payments-receivable-data
   • Export CSV                   → GET /info/history/payments-receivable-export
